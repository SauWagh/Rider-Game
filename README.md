# Hyper Rider

A fast-paced racing game developed using Unity and C# that challenges players to complete levels while collecting stars and avoiding obstacles.

## Play Online

Play the game on itch.io: https://saurabh14.itch.io/hyper-rider

## Download the latest build:
https://drive.google.com/file/d/1KdTddN4Iv43yXbrE-PA05C8vi3XAmYBH/view?usp=drive_link

## Features

- 10 progressively challenging levels
- Star collection system to unlock new levels
- Each level requires collecting a specific number of stars to progress
- Smooth vehicle controls using Left Arrow and Right Arrow keys
- Neon-inspired visual design
- Score and level progression system
- Responsive and engaging gameplay mechanics

## Controls

| Key | Action |
|------|----------|
| ← Left Arrow | Move Left |
| → Right Arrow | Move Right |
